##天天看 逆向了哪些软件？
起初逆向了快播，后又逆向了西瓜、吉吉等，逆向资源有很多好处，比如减轻服务器的压力。

##天天看 可以逃过法网么？
什么是法律？法律明文禁止则不可做，没有明文便可做。所以大家都可以做在线播放器创业。
